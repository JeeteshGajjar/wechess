
# WeChess

This app is an online chess game application.

<img src="demo.gif">

## Setting Up

* make sure you have Meteor installed
* clone this repo

```
npm install
npm start
```
Your app should be running [http://localhost:3000](http://localhost:3000)


## deploy


### deploy to AWS 

```
npm install -g mupx

mupx setup
mupx deploy

```


### deploy to modulus

```
npm deploy

```
